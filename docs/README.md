# 文档索引

这个目录用于存放 AIDbOptimize 的长期设计文档、执行计划和任务清单。

当前约定：

- `docs/plans/`：分阶段计划与执行清单
- `docs/plans/active/`：正在推进或即将推进的计划
- 每个计划使用一个独立文件夹，便于后续继续追加设计、复盘和状态变更

推荐约定：

- `design.md`：描述总体目标、架构方向、边界和约束
- `detailed-design.md`：描述详细设计，包含数据结构、类设计、接口、页面草图、伪代码
- `tasks.md`：描述可执行任务清单，适合后续 agent 逐项勾选和回填状态
